Développement et maintenance d'applications web

Description

Le projet consistait à participer au développement d'une application avec les outils Grist et Bubble. Je suis intervenu sur certaines parties du projet sans en assurer l'intégralité. J'ai également réalisé la correction de bugs sur une application existante développée en JavaScript.

Technologies utilisées

  • Langages : JavaScript
  • Outils : Grist, Bubble

Cadre de réalisation

Stage réalisé chez Audet Conseil, une entreprise spécialisée dans le conseil et le développement de solutions numériques.
